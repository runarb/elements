Broadcast

Broadcast automatically fetches broadcast messages for the current url.

Uses f-toast under the hood.

Props

Refetch interval: interval — default 300 000 (5 minutes)

Environment: dev — default false

Url to check for: url — default window.location.href

Example usages

// Fetches from dev.finn.no instead of finn.no// 4 second refetch interval// The url is automatically set through window.location.href, but you can override this by passing url as a param
